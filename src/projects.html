Trip App Final Project
This trip app was part of my final exam for my Android Application Development class. It uses Fragments to allow users to select which items they wish to take with them on their trip. As you can see from the video, the users inputted data is passed through out the fragments until the "Exit" button is clicked on the last Fragment which will terminate the app.

RewardMe Side Project
RewardMe is a web application that uses a REST API microservice for the backend (page with the black background) that connects to the h2 database, and Springboot and thymeleaf microservice for the front end that accesses the database information through the REST API. As shown in the video, users can enter their own rewards in additon to the default rewards. The application will then give users a random reward.
